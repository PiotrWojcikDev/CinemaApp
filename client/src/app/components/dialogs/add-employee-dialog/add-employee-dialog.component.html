Dodawanie pracownika
*Imię jest wymagane.
*Nazwisko jest wymagane.
*Email jest wymagany. *Email jest niepoprawny.
*Numer telefonu jest wymagany. *Numer telefonu jest niepoprawny.
*Pensja jest wymagana.
*Hasło jest wymagane. *Hasło musi zawierać przynajmniej: 8 znaków, wielkie i małe litery, cyfrę oraz znak specjalny.
*Potwierdzenie hasła jest wymagane. *Hasła nie są jednakowe.
